How x4 Encryption Works

A sophisticated multi-layer approach to unbreakable encryption

01

Input & Validation

Enter your message and two strong passwords. Text is normalized using NFKC for consistency.

plaintext ‚Üí UTF-8 ‚Üí NFKC normalization
02

PBKDF2 Key Derivation

Each password undergoes PBKDF2 hardening with configurable iterations (120K-1.2M) and salt.

password + salt + context ‚Üí PBKDF2 ‚Üí AES key
03

Layer 1-2 Encryption

First password encrypts data twice with unique salts and IVs for each layer.

Layer 1: AES-GCM(data, key1, iv1, AAD) Layer 2: AES-GCM(layer1, key2, iv2, AAD)
04

Layer 3-4 Encryption

Second password adds two more encryption layers with independent parameters.

Layer 3: AES-GCM(layer2, key3, iv3, AAD) Layer 4: AES-GCM(layer3, key4, iv4, AAD)
05

Output Generation

Final ciphertext is encoded in Base64 URL-safe format with self-describing header.

x4v2|k=pbkdf2;i=600000;h=SHA-256||[ciphertext]
